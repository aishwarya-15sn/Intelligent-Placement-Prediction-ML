## Project Objective

The primary goal of this project is to develop a **data-driven system capable of assessing job readiness and predicting placement probability** based on multiple academic and skill-based attributes.

This system can assist:

• Students in understanding employability readiness  
• Institutions in improving placement preparation strategies  
• Placement cells in identifying skill gaps among students
